# 🎮 MCO Scratch Downloader

A simple and powerful web tool to download publicly shared Scratch projects as real `.sb3` files directly from your browser.

---

## 🚀 About This Project

MCO Scratch Downloader is developed by **Master Coding Organisation (MCO)** and created by **Sidiq Ul Akbar**.

This tool allows users to:
- Download Scratch projects using a link or project ID
- Get complete `.sb3` files (including sprites, scripts, and sounds)
- Use it directly in the browser without installing anything

---

## ⚙️ How It Works

The downloader uses a browser-based library to:
1. Fetch project data from Scratch servers
2. Collect all assets (images, sounds, scripts)
3. Package everything into a `.sb3` file
4. Trigger a download instantly

## 📌 How to Use

1. Open the website  
2. Paste a Scratch project link or ID  
3. Click **Download**  
4. Your `.sb3` file will be downloaded  

---

## 🚧 Limitations

- Cannot download private/unshared projects  
- Large projects may take longer  
- Depends on Scratch API availability
